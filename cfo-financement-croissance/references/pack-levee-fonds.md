# Pack levée de fonds

Documents à préparer pour une levée de fonds equity (BA, seed, série A+).

## 1. Teaser (2 pages)

Document court envoyé en 1ère approche aux investisseurs.

Contenu :
- Société : nom, SIREN, fondateurs
- Pitch : problème + solution + traction
- Marché : TAM / SAM / SOM
- Modèle économique
- Métriques clés (ARR, growth, burn, runway)
- Tour : montant recherché, valorisation cible
- Use of funds
- Contact

Template : `templates/teaser-investisseurs.md`.

## 2. Pitch deck (15-25 slides)

Structure standard :

1. Couverture
2. Problème
3. Solution
4. Marché (TAM/SAM/SOM)
5. Produit (démo / screenshots)
6. Traction (chiffres clés, croissance)
7. Business model + unit economics
8. Go-to-market
9. Concurrence + différentiation
10. Roadmap produit
11. Équipe (fondateurs + key hires)
12. Financials (P&L 5 ans + cash)
13. Tour : montant, use of funds, milestones
14. Demande (montant, valorisation cible)
15. Annexes / Q&A potentielle

Outils : Pitch.com, Beautiful.ai, ou Keynote / Google Slides.

## 3. Business plan détaillé (cf. cfo-budget-forecast)

- 5 ans P&L + Bilan + Cash flow
- 3 scénarios
- Hypothèses détaillées
- Sensitivity sur les variables clés

## 5. Data room

Référence : `templates/data-room-checklist.md`.

Plateformes : Drooms, Intralinks, Datasite, Box, Google Drive (sécurisé).

Sections typiques :
- 01. Documents juridiques (statuts, PV, conventions)
- 02. Financials (états, audit, fiscal)
- 03. Commercial (contrats clients, pipeline)
- 04. Produit / IP (brevets, code base, doc tech)
- 05. RH (contrats, organigramme, plans BSPCE)
- 06. Litiges / risques
- 07. Tech infrastructure
- 08. Compliance

## 6. Memorandum d'investissement (optionnel pour rounds > 5 M€)

Document long (30-50 pages) détaillant :
- Société
- Marché
- Stratégie
- Financials détaillés
- Risques
- Tour proposé

Souvent rédigé avec l'aide d'un advisor ou banquier d'affaires (boutiques M&A, ex: Cambon Partners, Lazard Tech...).

## Cycle de levée typique (Série A)

| Mois | Action |
|------|--------|
| -12 | Préparation pitch + métriques + intros warm |
| -9 | First meetings VCs |
| -6 | Approfondissements, démos, rencontre équipe |
| -4 | Term sheets reçus |
| -3 | Négociation termes |
| -2 | Due diligence (légale, financière, tech, RH) |
| -1 | Documentation juridique (Pacte, Statuts, BSA...) |
| 0 | Closing + virement |

Total : 9-12 mois.

## Term sheet

Document non engageant fixant les principaux termes :

- **Valorisation pre-money / post-money**
- **Type d'actions** (préférence simple, participating, anti-dilution)
- **Liquidation preference** (1x, 2x, 3x, capped, uncapped)
- **Anti-dilution** (full ratchet, weighted average broad/narrow base)
- **Pre-emption rights** (ROFR, ROFO)
- **Drag-along, tag-along**
- **Board** : nb sièges, observateurs
- **Vesting fondateurs** (cliff, durée)
- **Pool BSPCE** : %
- **Reps & warranties**
- **Closing conditions**
- **Exclusivité** (4-8 semaines pour le VC lead)

→ **Important** : faire valider par avocat business spécialisé M&A (avocat dédié aux fondateurs, pas du VC).

## Due diligence

### Légale

- Statuts, PV, conventions, contrats clés
- Conformité réglementaire
- Litiges actuels et passés
- IP (brevets, marques)

### Financière

- États financiers + audit
- Forecast + business plan
- Working capital
- Tax

### Tech

- Architecture + scalabilité
- Sécurité
- IP du code
- Dette technique

### RH

- Contrats top managers
- BSPCE / vesting
- Conventions collectives
- Litiges sociaux

### Commercial

- Contrats clients top
- Pipeline + conversion
- Pricing

## Closing

Au closing :
- Augmentation de capital (CA + AGE notariée)
- Signature pacte d'actionnaires
- Mise à jour cap table
- Virement des fonds

Coût closing : avocat 30-100 k€ HT + frais notaire + commission cabinet M&A si advisor.

## Adaptation par audience

**Mode EC** : accompagnement de la levée pour le client (souvent en lien avec un avocat business). Vous préparez les financials et challengez la valorisation.

**Mode PME** (fondateur) : c'est VOTRE levée. Recruter un **advisor expérimenté** (ex-VC, banquier d'affaires) si tour > 5 M€. Avocat business obligatoire dès 1 M€.

## Erreurs fréquentes à éviter

- ❌ Pitch deck sans démo / preuves de traction
- ❌ Financials trop optimistes (hockey stick invraisemblable)
- ❌ Mauvais choix de VCs (pas alignés avec stade / secteur)
- ❌ Term sheet signé sans avocat
- ❌ Cap table illisible pour les investisseurs
- ❌ Data room incomplète au moment de la DD (delays)
