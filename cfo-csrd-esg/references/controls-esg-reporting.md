# Internal controls over ESG reporting

CSRD impose un niveau d'**audit-grade rigor** sur les données ESG, équivalent au reporting financier.

## Pourquoi des contrôles ESG ?

- Les données ESG sont **éparpillées** (énergie, RH, achats, supply chain...)
- Beaucoup d'estimations / hypothèses (Scope 3 surtout)
- Risque de **greenwashing** si non contrôlé
- L'auditeur sustainability vérifie les contrôles (limited assurance dès 2025, reasonable à terme)

## Cadre méthodologique

### COSO Internal Control extended to sustainability

Application des 5 composantes COSO (cf. cfo-risques-conformite) au reporting ESG :

1. **Environnement de contrôle** : tone at the top sur ESG
2. **Évaluation des risques** : risques de non-conformité ESG
3. **Activités de contrôle** : procédures, validations, séparation des fonctions
4. **Information & communication** : flux des données ESG
5. **Pilotage** : surveillance continue

### Standards spécifiques

- **ISSA 5000** (International Standard on Sustainability Assurance) : standard international d'audit ESG
- **SSAE 3000** : équivalent US

## Activités de contrôle clés

### 1. Procédures formalisées

Pour chaque indicateur ESG :
- Owner (rôle)
- Fréquence
- Source données
- Méthode de calcul
- Validation

### 2. Séparation des fonctions

- Collecte ≠ validation ≠ consolidation
- Exemple : opérationnel collecte → manager fonction valide → CSO consolide → CFO challenge → Comité ESG approuve

### 3. Réconciliations périodiques

- Données ESG vs données financières (cohérence)
- ESG inter-périodes (variations expliquées)
- ESG inter-sites (anomalies détectées)

### 4. Documentation des hypothèses

Pour chaque calcul :
- Hypothèses retenues
- Justification
- Source (étude, base de données, expertise)
- Date

Exemple : Scope 3 catégorie "achats biens et services" → hypothèse facteurs d'émission monétaires Base Carbone ADEME version 23.x.

### 5. Pistes d'audit

Pour chaque donnée publiée :
- Source originale (facture, attestation, calcul)
- Étape de collecte
- Étape de validation
- Étape de consolidation
- Date + nom des intervenants

### 6. Tests de cohérence

- Variations YoY (alerter si > X%)
- Cohérence entre indicateurs (ex: si effectif ↑ et émissions Scope 1 ↓, expliquer)
- Comparaison benchmarks sectoriels

## Maturité contrôle ESG

| Niveau | Description |
|--------|-------------|
| 1. Ad hoc | Données collectées au coup par coup, par personne (vulnerable au turnover) |
| 2. Reproductibles | Procédures écrites, mais inégalement appliquées |
| 3. Définis | Procédures formalisées, formation, application uniforme |
| 4. Pilotés | KPI sur la qualité des données, revue régulière |
| 5. Optimisés | Outil dédié, automatisation, pistes d'audit complètes |

PME début ESG : niveau 1-2.
ETI mature : niveau 3-4.
Auditeur exige niveau 3+ pour limited assurance.

## Tests internes (par auditeur interne ou CFO)

### Tests à faire annuellement

Sur chaque indicateur ESG matériel :

1. **Test de complétude** : toutes les sources ont été collectées ?
2. **Test d'exactitude** : les calculs sont corrects ?
3. **Test de séparation** : pas double comptage entre Scope 2 et 3 par exemple ?
4. **Test de cohérence** : l'évolution YoY est explicable ?
5. **Test de validation** : les multi-niveaux sont appliqués ?

Échantillonnage : 10-25 enregistrements par indicateur.

### Reporting des résultats

- Résumé executive
- Détail par indicateur
- Findings + remédiation
- Plan d'amélioration N+1

## Coordination avec audit financier

Le CAC (commissaire aux comptes) **valide les données financières** + **les liens avec ESG** :
- Cohérence chiffres ESG monétaires (ex: investissement décarbonation €) avec balance comptable
- Risques climat reflétés dans les provisions / dépréciations
- Engagements hors bilan (objectifs SBTi, etc.)

L'**auditeur sustainability** (peut être le même cabinet ou différent) audite les données ESG.

→ Coordination essentielle entre les 2 missions.

## Avertissement

Les contrôles ESG sont **un projet de transformation** sur 12-18 mois. La 1ère année de reporting CSRD est **toujours un challenge** — accepter une certaine imperfection, viser l'amélioration continue.

Pour mise en place : cabinet sustainability + auditeur en amont (pas seulement post-rapport).
