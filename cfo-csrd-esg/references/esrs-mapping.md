# ESRS, European Sustainability Reporting Standards (Set 1)

10 standards thématiques + 2 standards transverses (ESRS 1 et ESRS 2).

Source : EFRAG (https://www.efrag.org/en/sustainability-reporting).

## Standards transverses (obligatoires si in-scope)

### ESRS 1, General Requirements

Cadre général : périmètre, principe de double matérialité, référentiels, formats.

### ESRS 2, General Disclosures

Informations transverses obligatoires :
- Stratégie (BP1 à BP6)
- Gouvernance (GOV-1 à GOV-5)
- Gestion des risques (IRO-1, IRO-2)
- Politiques (MDR-P)
- Actions (MDR-A)
- Targets et metrics (MDR-T, MDR-M)

## Standards Environment (E)

### E1, Climate Change

**Obligatoire pour tous les in-scope** (présomption de matérialité).

Contenu :
- Plan de transition climat (objectifs alignés Paris : neutralité 2050, -55% en 2030)
- Émissions GES Scope 1, 2, 3
- Risques climat (transition + physique)
- Métriques : intensité carbone (tCO2e/CA), trajectoire SBTi
- CAPEX & OPEX décarbonation
- Liens avec taxonomie verte (alignment)

### E2, Pollution

Si matériel :
- Pollution air / eau / sol
- Substances préoccupantes / extrêmement préoccupantes (CMR, REACH)
- Microplastiques
- Plan d'action pollution

### E3, Water and Marine Resources

Si matériel :
- Consommation eau (m³/an)
- Stress hydrique des sites (zones water-stressed)
- Pollution eau
- Plan d'action eau

### E4, Biodiversity and Ecosystems

Si matériel :
- Sites en zones sensibles biodiversité
- Impacts biodiversité (déforestation, fragmentation habitats)
- Espèces menacées impactées
- Plan d'action biodiversité

### E5, Resource Use and Circular Economy

Si matériel :
- Consommation ressources (matières premières)
- Inflows / outflows matières
- Déchets produits (recycled / non-recycled)
- Économie circulaire (design produit, réparabilité, recyclage)

## Standards Social (S)

### S1, Own Workforce

Si matériel (presque toujours) :
- Effectif total + breakdown (par genre, âge, contrat, géographie)
- Politique salariale (égalité salariale F/H, écart médian)
- Conditions de travail
- Santé / sécurité (taux fréquence accidents, journées perdues)
- Formation (heures par employé)
- Représentation collective (nombre de comités, taux syndicalisation)
- Diversité, inclusion
- Cas de discrimination / harcèlement traités

### S2, Workers in the Value Chain

Si matériel :
- Travailleurs en amont / aval (sous-traitants, fournisseurs, prestataires)
- Risques droits humains / travail forcé / enfants
- Devoir de vigilance (cf. loi française 2017)
- Audit fournisseurs (% audités, résultats)

### S3, Affected Communities

Si matériel :
- Communautés locales impactées par les opérations
- Engagements communautaires
- Conflits / réclamations

### S4, Consumers and End-Users

Si matériel :
- Sécurité produits / services
- Pratiques marketing responsable
- Protection données personnelles
- Inclusion / accessibilité

## Standard Gouvernance (G)

### G1, Business Conduct

Si matériel (presque toujours) :
- Code de conduite
- Politique anti-corruption + cas
- Lanceur d'alerte (whistleblower)
- Lobbying (transparence)
- Pratiques fournisseurs (paiement à temps, etc.)
- Activités politiques

## Cadre d'application

### Étape 1, Double matérialité

Pour chaque standard E, S, G : évaluer si le sujet est matériel **financièrement** ou **en termes d'impact**.

→ Si matériel sur **au moins un des deux axes**, le standard doit être **complètement reporté**.

### Étape 2, Disclosures requises

Pour chaque standard matériel :
- Disclosures obligatoires (X dans le standard)
- Métriques spécifiques (M-1, M-2, ...)
- Targets si fixés
- Politiques + actions

### Étape 3, Format

Le rapport sustainability est intégré au **rapport de gestion** (L. 225-102-1 C. com.).
Format **xHTML** pour digitalisation (ESEF, European Single Electronic Format) à terme.

## Adaptation PME (ESRS-LSME)

Pour les PME cotées Wave 3, version simplifiée disponible :
- Moins de disclosures obligatoires
- Format simplifié
- Métriques limitées aux essentielles

## Adaptation par audience

**Mode EC** : aide à la mission "préparation CSRD" pour le client. Mapping initial des disclosures applicables.

**Mode PME / ETI** : démarche structurée 12-18 mois. Identifier les standards matériels avant de tout documenter.

## Workflow recommandé

1. Lecture rapide des 10 standards (1 jour)
2. Workshop double matérialité (2-3 jours, équipe + parties prenantes)
3. Sélection des standards matériels (sortie workshop)
4. Pour chaque standard matériel : disclosure mapping + collecte data
5. Préparation premier rapport (3-6 mois)
6. Audit sustainality (limited assurance)

## Avertissement

ESRS sont **techniques**. Pour la mise en œuvre concrète : **expert sustainability** indispensable. Risque de mauvaise interprétation des disclosures = audit qui échoue.
