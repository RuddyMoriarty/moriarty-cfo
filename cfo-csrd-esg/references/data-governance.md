# ESG Data Governance — modèle CFO/CSO co-leadership

## Le défi des données ESG

CSRD impose un niveau d'**audit-grade rigor** sur les données ESG, équivalent au reporting financier.

Mais les données ESG sont **éparpillées** :
- Énergie : facture EDF
- Carburants : factures véhicules
- Déchets : prestataire collecte
- Voyages : agence de voyage
- Effectif : SIRH
- Formation : LMS
- Accidents : registre santé / sécurité
- Achats : ERP
- Supply chain : Excel fournisseurs

→ Sans gouvernance structurée, impossible de produire un rapport fiable + audité.

## Modèles d'organisation

### Modèle 1 — CFO seul ownership (33% des entreprises)

Avantages :
- Discipline financial reporting
- Cohérence avec rapport de gestion
- Audit-ready

Inconvénients :
- Manque d'expertise sustainability
- Risque de minimiser sujets non quantifiables

### Modèle 2 — CSO seul ownership (32%)

Avantages :
- Expertise sustainability
- Engagement parties prenantes
- Veille réglementaire

Inconvénients :
- Manque de discipline reporting
- Coordination difficile avec finance

### Modèle 3 — Co-leadership CFO + CSO (35%)

**Source EFRAG 2024 : modèle le plus répandu.**

Avantages :
- Combine expertise + discipline
- Reporting intégré (financier + sustainability)
- Crédibilité auprès des auditeurs

Inconvénients :
- Coordination requise
- Coût (2 personnes mobilisées)

→ **Recommandation pour les ETI in-scope** : co-leadership avec rôles définis.

## Répartition des rôles co-leadership

### Le CFO apporte

- **Discipline reporting** : cycles, deadlines, contrôles
- **Cohérence financière** : équivalence avec reporting financier
- **Audit-ready** : pistes d'audit, justificatifs, contrôles internes
- **Reporting intégré** : sustainability dans le rapport de gestion
- **Risques financiers ESG** : impact financier des risques climat / supply chain

### Le CSO apporte

- **Expertise sustainability** : connaissance ESRS, GHG Protocol, etc.
- **Engagement parties prenantes** : workshops, dialogues, plaintes
- **Veille réglementaire ESG** : CSRD, taxonomie verte, France 2030
- **Pilotage transformation** : décarbonation, économie circulaire
- **Communication externe** : rapport sustainability, événements, presse

## Comité ESG / Sustainability Committee

Composition typique :
- CFO + CSO (co-pilotes)
- DG / DG délégué (sponsor)
- DRH (S1 workforce)
- Direction Industrielle / Opérations (E1, E2, E3)
- Direction Achats (S2 supply chain)
- Direction Juridique (G1)
- Représentant board (membre indépendant)
- Représentants salariés (CSE)

Cadence : mensuel pendant la phase de mise en route, trimestriel ensuite.

### Approche "Excel + Sharepoint"

Pour démarrer (PME, première année) :
- Tableurs structurés par sujet
- Process de collecte mensuelle / trimestrielle
- Validation manuelle par responsable
- Stockage Sharepoint / Drive

Limites : pas scalable au-delà de la 1ère année, audit difficile.

## Procédures de collecte

### Étape 1 — Cartographie des sources

Pour chaque indicateur ESG :
- D'où vient la donnée ?
- Qui en est responsable ?
- Quelle fréquence de collecte ?
- Quelle qualité ?

### Étape 2 — Procédure formalisée

Pour chaque indicateur :
- Qui collecte (rôle, pas une personne)
- Comment (template, source)
- Quand (deadline)
- Comment vérifier (contrôle)

### Étape 3 — Validation multi-niveaux

- Niveau 1 : collecte par opérationnel
- Niveau 2 : revue par responsable de fonction
- Niveau 3 : consolidation par CSO/CFO
- Niveau 4 : validation finale par Comité ESG

### Étape 4 — Pistes d'audit

Pour chaque donnée publiée :
- Source originale (facture, attestation, calcul)
- Hypothèses retenues
- Validation
- Date

## Indicateurs prioritaires à mettre en place

### Année 1 (priorité absolue)

**E1 climat** :
- Consommation énergie (électricité kWh, gaz m³, autres combustibles)
- Carburants flotte véhicules (litres essence, diesel, électrique kWh)
- Voyages employés (km train, avion, voiture)
- → Calcul Scope 1 + 2 + premières estimations Scope 3

**S1 workforce** :
- Effectif (CDI, CDD, alternance, freelances)
- Diversité (F/H par fonction)
- Heures formation (par employé)
- Accidents du travail (taux fréquence + gravité)
- Turnover

**G1 governance** :
- Code de conduite (date dernière révision)
- Cas d'éthique (nb signalés, nb traités)
- Politique anti-corruption (oui/non)

### Année 2

Ajouter :
- E2 pollution (si site industriel)
- E5 économie circulaire (déchets, recyclage)
- S2 supply chain (audits fournisseurs)
- S4 consumers (incidents qualité)

### Année 3+

Compléter selon matérialité confirmée.

## Coordination avec la Comptabilité

Le CFO doit s'assurer que :
- Cycle de reporting ESG aligné avec cycle financier (mêmes deadlines, mêmes périodes)
- Données ESG monétaires (ex: investissement décarbonation €) cohérentes avec balance comptable
- Engagements ESG intégrés au business plan (cf. cfo-budget-forecast)
- Risques ESG intégrés à la cartographie risques (cf. cfo-risques-conformite)

## Avertissement

La data governance ESG est **un projet de transformation** sur 12-18 mois minimum. Pour démarrer, recruter ou former un **CSO** est souvent le 1er investissement.

Pour les PME : **co-pilotage CFO + responsable QSE / RSE** existant peut suffire.
